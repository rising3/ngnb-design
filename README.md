## Requirements

- Node.js 18 or higher

## How to build from source

### prerequisites

node.js, npm, git need to be installed.

```sh
git clone https://github.com/rising3/ngnb-design.git
cd ngnb-design
npm i
npm run build
```

To run the example:
```
npm run dev
```
